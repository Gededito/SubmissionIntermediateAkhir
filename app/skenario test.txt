1. Pengujian Login View Model :
    - Memastikan data tidak null
    - Memastikan saat berhasil login dan mengembalikan Result.Success
    - Memastikan data dugaan sesuai dengan yang diharapkan
    - Memastikan mengembalikan Result.Error
    - Memastikan fungsi saveUser() pada repository terpanggil
    - Memastikan sukses logout

2. Pengujian Register ViewModel :
a. Ketika pengguna berhasil register
 - Memastikan bila register berhasil dan mengembalikan Result.Success
 - Memastikan data tidak null
b. Ketika pengguna mengalami masalah koneksi
 - Memastikan live data akan mengembalikan Result.Error
 - Memastikan data tidak null

3. Pengujian Add Story View Model
- Memastikan add story data tidak null.
- Memastikan mengembalikan Result.Success.
- Memastikan data yang dikembalikan sesuai yang diharapkan

4. Pengujian Maps View Model
- Memastikan bahwa maps mengembalikan Result.Success
- Memastikan Maps sesuai dengan yang diharapkan
- Memastikan data yang dikembalikan sesuai yang diharapkan

5. Pengujian Story View Model
- Memastikan bahwa paging mengeluarkan data dengan benar
- Memastikan data tidak null
- Memastikan data yang dikembalikan sesuai yang diharapkan
